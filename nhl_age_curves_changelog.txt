v0.24.0 - KNN Full-Career Matching & Era Multiplier Corrections
---------------------------------------------------------
- CHANGED: KNN clone matching now uses the player's entire career instead of last 3 seasons.
  Previously tail(3) meant a player with an unusual recent 3-year stretch (injury recovery,
  lockout, line change) would match to players with similar recent blips rather than similar
  career shapes. Full career matching provides more signal and produces better clone quality.
  Mid-season pacing still applies only to the last (current) season.
- CHANGED: Fallback slope calculation updated to use full career length (len(career_df) - 1)
  instead of len(recent) - 1 = 2, giving a more stable average annual trajectory estimate.
- FIXED: Era multiplier for 1968-1979: 0.85 → 0.89.
  Actual NHL GF/GP that era was ~3.40/team (not 3.6+). 3.05/3.40 = 0.897.
  0.85 was over-deflating Esposito, Orr, Lafleur, Trottier era stats.
- FIXED: Era multiplier for 2013-2017: 1.10 → 1.12.
  Analytics/trap era averaged ~2.72 GF/GP. 3.05/2.72 = 1.121. Minor but evidence-based.
- CHANGED: Era multiplier for 2005-2012: 1.05 → 1.06.
  Post-lockout average ~2.87 GF/GP. 3.05/2.87 = 1.063. Marginal correction.
- CONFIRMED: All other multipliers (0.80, 0.90, 1.15, 1.00) verified within tolerance.

v0.26.0 - Peak Ranking Fallback & KNN Elite Tier Matching
---------------------------------------------------------
- FIXED: Peak Season Ranking now works even when the player's PlayerID is absent
  from the parquet. Uses the player's known raw peak stat value (stored as
  raw_peak_val in peak_info and passed as customdata[5]) to count how many

  fillna(0).mean() in the projection loop with dropna().mean(), so only clones who
  actually played at each future age set the trend. Early retirees no longer flood
  the average with zeros, and long-career greats (Jagr, Chelios) properly anchor
  the late-career tail.

v0.29.0 - Smoother KNN Projection Decline
---------------------------------------------------------
- CHANGED: pct_change clamp tightened from ±0.5 to [-0.12, +0.25]. Old ±50% annual
  limit allowed unrealistic cliff drops for elite players (e.g. McDavid 100→50 pts
  in a single projected year). Max 12% annual decline is still a meaningful decay
  while respecting that top players age more gracefully than average clones suggest.
- CHANGED: Clone average now blended 70% raw mean / 30% previous year's avg before
  computing pct_change. Late-career clone pools shrink to 3-4 players, making one
  injury year whipsaw the mean. The blend damps that noise without biasing the trend.
- CHANGED: Fallback extrapolation rate floor tightened from -0.12 to -0.08 to stay
  consistent with the live-clone clamp.

v0.29.1 - Baseline Cliff Fix at Ages 38-40
---------------------------------------------------------
- FIXED: NHL 75th Percentile Baseline crashed to 0 at ages 38-40. Root cause: very
  few players in the parquet played 40+ GP at those ages, so the raw 75th percentile
  of that sparse pool collapsed to near-zero.
- ADDED: Complementary decay cap to the survivorship rule — if the baseline drops
  more than 15% year-over-year after age 31, it is clamped to 0.85× the previous
  year (max 15% annual drop). This mirrors the existing rise cap (0.92×) and keeps
  the baseline visible and realistic through age 40.

v0.35.0 - Goalie Era Adjustment
---------------------------------------------------------
- ADDED: get_goalie_era_sv_offset(year) — returns additive Save% offset (0-1 scale) to
  normalize goalie Save% to the 2018+ league-average baseline (0.9110). Uses an 8-period
  lookup table of historical NHL league-average Save% values (0.871 expansion era to
  0.912 analytics era peak). Preserves each goalie's deviation above/below their era mean.
- ADDED: Goalie era adjustment block in player pipeline (NHL rows only, pre-groupby).
  Targets WeightedGAA and WeightedSV (the GP-weighted sums), not the post-groupby derived
  columns Save % and GAA, which don't exist in raw_df at adjustment time.
  - GAA: WeightedGAA * era_multiplier (same GF/GP multipliers as skaters — GAA scales
    directly with league scoring; 3.50 in 1985 x 0.80 = 2.80 normalized).
  - Save%: WeightedSV + (sv_offset * 100 * GP) — offset scaled to WeightedSV units.
  - Shutouts: raw / era_multiplier — inverse mult gives more credit for high-scoring eras
    (10 SO in 1985 / 0.80 = 12.5 adjusted).
  - Wins: not adjusted (too team-dependent for reliable era normalization).
- UPDATED: apply_era_to_hist() now accepts is_goalie=False. Goalie branch adjusts GAA,
  Save% (0-1 scale in hist_df), and Shutouts in historical parquet data so KNN clone
  matching stays consistent with the displayed era-adjusted player curve.
- UPDATED: apply_era_to_hist call site passes is_goalie=is_goalie.
- Era-Adjust toggle was already visible for goalies; now it actually does something.
